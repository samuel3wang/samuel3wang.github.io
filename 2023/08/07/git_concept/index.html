發表於 2023-08-06 更新於 2023-08-26

Branch

Local

Create Local Branch

Switch to Branch
git checkout <branch-name>

Delete Local Branch
git branch -d <branch-name>

Remote

Create Remote Branch
git push origin <branch-name>

Merge

Merge Branch
git merge <branch-name>

Local 需要透過 git pull 來更新 Local 的資料
git pull
