One-command Deploy on GitHub

  1. 安裝 hexo-deployer-git

  2. 在GitHub上建立一個新的repository，名稱為 .github.io
    這裡的 是GitHub帳號名稱，因為跳過這一步驟一直卡在404

  3. 在 _config.yml 設定 deploy

  4. Deploy command
